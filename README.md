# Marcelo's Home Assistant Add-ons

Repositório pessoal de add-ons interessantes para o [Home Assistant](https://www.home-assistant.io/).

## Instalação

1. No Home Assistant, vai a **Definições → Add-ons → Add-on Store**.
2. Clica no menu (⋮) no canto superior direito → **Repositories**.
3. Cola o URL deste repositório e clica em **Add**:

   ```
   https://github.com/MarceloRRMartins/hassio-addon
   ```

4. Os add-ons abaixo passam a aparecer na loja.

## Add-ons disponíveis

| Add-on | Descrição |
| ------ | --------- |
| [Threadfin](./threadfin) | Proxy M3U e gestor de EPG para Plex DVR, Emby e Jellyfin Live TV. |
